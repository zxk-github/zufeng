### ts基本数据类型
1. 字符串类型
```
  let str: string = 'a'
```
2. 数字类型
```
  let num: number = 0;
```
3. 布尔类型
```
  let bool: boolean = true;
``
4. 数组类型
```
  let arr1 = number[] = [1,2,3];

  let arr2 = Array<string> = ['a', 'b']
```

5. 元组类型
概念: 表示一个已知类型和数量的数组

```
  let info: [string, number] = ['a', 1]
```
